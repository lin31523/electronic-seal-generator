# 电子印章生成器

> **特别声明：本工具生成模板仅供学习参考，请勿用于非法用途，否则后果自负**

本项目不提供电子签章合规能力，不包含 CA 证书、身份认证、法律效力校验或防伪认证能力。使用者需自行承担使用风险。

一个纯浏览器本地运行的电子印章 PNG 生成工具。所有参数处理和图片导出都在浏览器中完成，不会上传数据到服务器。

## 功能特性

- 主体文字、防伪码、中心内容、抬头文字配置
- 支持圆章、椭圆、方章
- 支持单线条 / 双线条边框
- 支持常用中文字体选择
- 支持文字大小、弧度、间距、边距、加粗
- 支持印章颜色、印泥浓度、做旧纹理
- 支持保存设置到浏览器本地
- 支持按主体文字命名导出透明 PNG

## 本地运行

```bash
npm install
npm run dev
```

打开浏览器访问：

```text
http://localhost:5174/
```

## 使用方法

1. 在“印章设置”中填写主体文字、防伪码、中心内容和抬头文字。
2. 选择章形、边框线型、印章大小、线条粗细和颜色。
3. 在“高级参数”中调整文字比例、弧度、间距、边距、加粗和做旧纹理。
4. 点击“保存设置”可将当前参数保存到浏览器本地。
5. 点击“导出透明 PNG”下载生成结果。

导出的文件名会优先使用主体文字，例如：

```text
旭格国际建材（北京）有限公司.png
```

如果主体文字为空，则默认导出为：

```text
电子印章.png
```
